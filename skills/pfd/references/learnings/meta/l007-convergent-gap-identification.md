# Learning 7: Consistent Gap Identification Across Rounds

**Insight:** R2 (pre-attentive content signaling) emerged as the unaddressed gap in every single round of analysis. Domain-focused personas proposed access mechanisms. PFD derivation identified the signal gap. When the same requirement surfaces independently across 4 different analytical frames, it's not a preference — it's a genuine perceptual constraint. This convergence is PFD's strongest validation signal. When running multiple derivations on the same problem, note which R[n] appear consistently — these are load-bearing.
