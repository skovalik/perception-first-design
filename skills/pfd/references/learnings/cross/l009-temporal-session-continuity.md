# Learning 9: Temporal/Session Continuity — PFD's Missing Dimension

**Insight:** PFD addresses what the user perceives NOW but not how they RE-ENTER a workspace or how RECENCY affects cognition. Personas with ATC, film editing, environmental psychology, accessibility, information architecture, and game UX lenses all independently identified this gap. A workspace visited after 48 hours is a different perceptual object than the same workspace visited 5 minutes ago — PFD has no layer for this. Candidate for formal L6 in v0.4 after testing. For now: note that PFD's 5 layers assume continuous interaction and need supplementation for interrupted/resumed sessions.
