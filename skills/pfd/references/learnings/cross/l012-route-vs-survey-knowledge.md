# Learning 12: Route vs Survey Knowledge in Spatial Tools

**Insight:** Siegel & White's spatial knowledge theory: users develop route knowledge (sequential, egocentric, fragile) before survey knowledge (allocentric, configurational, robust). Canvas tools should support BOTH stages. New users need route affordances (guided paths, breadcrumbs, landmarks). Experienced users need survey affordances (minimap, district overview, configurable views). PFD-S S1 (Depth-Attention Congruence) should distinguish between these knowledge types — the "appropriate density" depends on whether the user has route or survey knowledge of the space.
