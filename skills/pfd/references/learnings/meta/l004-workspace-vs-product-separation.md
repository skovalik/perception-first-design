# Learning 4: Workspace vs Product Separation

**Insight:** When personas analyze a specific workspace through PFD, they naturally propose workspace-level fixes (rearrange nodes, rename things, add visual zones). Many of these recommendations are actually product-level features the tool should provide. The separation between "user organizes content" and "tool provides perceptual infrastructure" is a critical design boundary. PFD should be applied at BOTH levels but the intervention target must be identified correctly. In marketing: distinguish "fix this specific email" (execution) from "fix the email structure" (system).
