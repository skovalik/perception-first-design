# Learning 5: Recursive Validation

**Insight:** When PFD is used to analyze a workspace that is ABOUT PFD, three levels operate simultaneously: (1) PFD as framework, (2) PFD applied to the workspace, (3) the workspace as proof-of-concept for the tool. When all three levels are legible, the workspace becomes its own best case study. This recursive structure is a strength, not a liability — but only when intentionally designed.
